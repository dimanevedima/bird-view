# Bird View Timer

Minimalist dark creative timer for music work.

Bird View alternates short work pulses with short empty spaces, helping you keep the whole track in view before going into pixel-level edits.

## Local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The app stores timer state, tiny task, and sessions in `localStorage` under `bird-view-timer-state`.

GitHub Pages is configured through `.github/workflows/pages.yml`.
